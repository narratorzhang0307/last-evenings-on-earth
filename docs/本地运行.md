# 本地运行说明

这份说明用于在本机打开“地球上最后的夜晚”，并让前端和照片服务连起来。

## 前端

在仓库根目录安装依赖：

```bash
npm install
```

启动页面：

```bash
npm run dev
```

默认访问地址：

```bash
http://127.0.0.1:3000
```

如果这个端口已经被占用，开发服务器会给出新的访问地址。

## 照片服务

进入服务目录：

```bash
cd server
```

安装服务依赖：

```bash
npm install
```

启动服务：

```bash
npm start
```

默认服务地址：

```bash
http://127.0.0.1:3008
```

## 连接前端和服务

前端默认可以在没有照片服务时运行；投稿会先尝试连接服务，失败时使用本地兜底。

需要连接服务时，在本地环境中设置：

```bash
VITE_API_BASE=http://127.0.0.1:3008
```

本地环境文件不要提交到仓库。

## 检查

检查前端类型：

```bash
npm run lint
```

检查生产构建：

```bash
npm run build
```

检查照片服务语法：

```bash
node --check server/server.mjs
```

## 常见情况

- 页面能打开但投稿没有写入服务：确认照片服务已经启动，并检查前端连接地址。
- 服务启动失败：确认端口没有被占用，或者在本地环境中换一个端口。
- 页面加载较慢：地球视图依赖三维场景，第一次加载会比普通页面稍慢。
- 投稿照片不显示：确认图片链接能公开访问，并以 `http` 或 `https` 开头。
